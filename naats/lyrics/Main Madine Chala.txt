Main madinay chala, mein Madinay chala,
Phir karam ho gaya main Madinay chala.

Kaif sa cha gaya main madinay chala,
Jhoomta jhoomta main madinay chala.

Mere Aaqa ka darr hoga paish-e-nazar,
Chahiye aur kya main Madine chala.

Mere Gande Qadam Aur Unka Haram
Laaj Rakhna Khuda Main Maine Chala

Gumbad-e-sabz par jab paray gi nazar,
Kya suroor aega main Madinay chala.

Sabz gunbad ka noor zang kar day ga dur,
Paye ga dil jila, main Madinay chala.

Ashk thamtay nahi, paon jamtay nahi,
Larkharaata hua main madinay chala.

Mere Siddique Umar ho salam aap par,
Aur rehmat sadaa main Madinay chala.

Woh uhod ki zammen jis k ander makeen,
Mere hamza piya main Madinay chala.

Unka gham chasham-e-tarr aur sauz-e-jigar,
Ab tou dai dai Khuda main Madinay chala.

Saaqia mai pila main madinay chala,
Mast-o-bekhud bana main Madinay chala.

Ae shajar a hajar tum bhi shams-o-qamar,
Daikho daikho zarra main Madinay chala.

Daikh taaray mujhe, yeh nazaray mujhe,
Tum bhi daikho zarra main Madinay chala.

Rooh-e-muztar theher tu nikalna udher,
Itni jaldi bhi kya main Madinay chala.

Haath uth’tay rahain mujhko daitay rahain,
Woh talab se siwa main Madinay chala.

Noor-e-haq k Hazoor apnay saray kasoor,
Bakhswanay chala main Madinay chala.

Woh Bakee ki zameen jis k ander makeen,
Mere Madani Zia main Madinay chala.

Un k minaar par jab paray gi nazar,
Kya saroor aega main Madinay chala.

Minbar-e-noor par jab uthay gi nazar,
Kya saroor aega main Madinay chala.

Dard-e-ulfat milay zauq barhnay lagay,
Jab chalay qaafila main Madinay chala.

Kya karega idher baandh rakht-e-safar,
Chal obaid-e-raza main Madinay chala.

Kya yeh tu ne kaha a Ubaid-e-Raza,
Soch tou kuch zarra main Madinay chala.

Main tou bus yoon hi tha main ki aukaat kya,
Qaafia yeh mila main Madinay chala.

Lutf tou jab milay mujh se murshid kahain,
Chal Obaid-e-Raza main Madinay chala.
